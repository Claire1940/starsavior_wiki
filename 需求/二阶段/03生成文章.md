一、查询资料：开启VPN模式
# 1. 搜集元数据（可以执行3次）
cd tools/content_pipeline && python3 collect.py --json ../keywords.json
# 2. 编辑 out/pending_review.json 筛选内容
# 3. 提取内容（可以执行3次）
python3 extract.py


二、生成文章
加上–test生成2篇，测试生成
cd tools/articles/modules/generation 
python3 generate-articles.py --test
如果都执行成功了，检查格式是否正确。
再去调用函数生成所有文章。
如果有一个失败，都要禁止执行后续所有文章。
python3 tools/articles/modules/generation/generate-articles.py
如果都执行成功了，检查格式是否正确：
1）python3 tools/clean_spaces.py content/ 请先运行修复表格行中，可能包含大量连续空格的错误
2）运行
rm -rf .next && npm run build 2>&1
和python3 tools/validate_mdx.py content/ --format json > tools/validation_report.json
扫描出所有格式错误，并立即手动确认、并修复所有错误（包括 补充缺失的FAQ）
